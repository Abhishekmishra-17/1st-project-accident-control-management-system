LANE DETECTION

The lane departure warning system uses cameras to determine if a vehicle has drifted across a lane. It sends audio, vibration, or visual notification if the vehicle has shifted across the lane. It includes two modules that are:-
a) First module warns the driver if the vehicle is leaving its lane.
b) Second module self drives the vehicle automatically inside the lane if the driver neglects the warning.
