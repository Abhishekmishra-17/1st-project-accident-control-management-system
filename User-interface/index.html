ACCIDENT CONTROL MANAGEMENT SYSTEM

ACMS helps us to construct a smart vehicle system with minimizing the limitations of existing methods and also enhancing the security of vehicles and human beings by reducing accidental injuries. Smart vehicle system will entail a speed and other parameters of vehicle sensing mechanism which automatically messages to personal contacts with the details of vehicle position when an accident occurs using the GPS system.
